# film-site-metadata-archive
本仓库用于整理与保存互联网上可公开访问的影视类网站元数据信息，重点在于站点层面的基础资料归档。整理内容包括站点名称、访问地址、内容分类、访问状态等客观字段，所有信息均来源于公开网络环境，仅用于信息汇总与长期留存。本仓库不涉及任何影视资源的存储、传播或解析行为，也不对所记录网站的内容质量、版权属性或更新频率作出判断。项目定位为中性的信息整理与索引工具，不构成任何形式的推荐、宣传或使用引导。
